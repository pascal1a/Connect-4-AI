**Connect-4-AI**

**Description**
  This program allows you to play a simple Connect-4 game against an AI in python. The AI is based on the minimax algorithm with a certain sscoring system. 


**Instructions**

    The program can be  executed online or locally on python. 
    
    For the online version use the following link:
    
    To run it locally the following steps are required:
      1. Make sure the following packages are avaiable: time, math, random and sys
      
      2.Install the pygame package (pip install pygame)
      
      3.Download the file Connect4.py and execute it
